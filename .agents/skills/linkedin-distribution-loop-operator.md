# linkedin-distribution-loop-operator

Use this skill to run a daily distribution cadence that compounds reach and
converts engagement signals into routed conversations.

## Required Inputs

- daily time budget
- ICP account watchlist
- active post inventory
- response SLA

## Daily Distribution Rhythm

- morning execution block:
  - publish
  - rapid comment replies
  - DM response pass
- evening execution block:
  - ICP engagement
  - signal follow-up
  - swipe-file capture

## Execution Sequence

1. Run morning block checklist.
2. Capture engagement signals:
   - comments
   - profile visits
   - follows
3. Route high-signal profiles to DM workflow.
4. Run evening block and relationship engagement.
5. Log daily metrics and next-day priorities.

## Output Contract

Return:

- daily_execution_log
- captured_signal_table
- routed_contacts
- next_day_priority_list

## Failure Conditions

- engagement work done without signal capture
- no routing path for high-signal contacts
- no end-of-day metric log
