# linkedin-authority-channel-flywheel

Use this skill to position personal and company brands as a clear authority and
connect LinkedIn to supporting channels without diluting focus.

## Required Inputs

- ICP definition
- positioning statement draft
- signature framework
- proof library
- channel list

## Positioning Formula

- who
- transformation
- method
- proof

## Execution Sequence

1. Draft authority positioning statement using formula.
2. Build signature framework:
   - named method
   - 3 to 5 components
   - before and after outcome
3. Build proof library requirements and gaps.
4. Design channel flywheel:
   - LinkedIn core
   - newsletter support
   - collaboration support
   - optional Instagram top-of-funnel use case
5. Define metrics for authority and pipeline progression.

## Output Contract

Return:

- final_positioning_statement
- signature_framework_map
- proof_library_checklist
- channel_flywheel_plan
- metric_dashboard_schema

## Failure Conditions

- vague positioning with no specific ICP
- no proof system behind authority claims
- channel plan with no link to pipeline outcomes
