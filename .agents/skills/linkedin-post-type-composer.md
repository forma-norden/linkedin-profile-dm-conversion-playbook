# linkedin-post-type-composer

Use this skill to compose high-performing posts from five reusable post types.

## Required Inputs

- selected content bucket
- selected post type
- one measurable claim or core point
- CTA objective

## Supported Post Types

1. lead magnet post
2. case study post
3. mistake post
4. framework post
5. contrarian truth post

## Execution Sequence

1. Generate hook options (3) aligned to post type.
2. Build body structure using post-type template.
3. Insert proof or evidence block if claim is outcome-based.
4. Select CTA pattern by objective:
   - lead capture
   - conversation
   - authority discussion
5. Validate readability and publish safety.

## Output Contract

Return:

- hook_options
- selected_post_structure
- final_post_draft
- cta_rationale
- quality_check_result

## Failure Conditions

- post type selected but template not applied
- outcome claim without evidence block
- CTA unrelated to post objective
